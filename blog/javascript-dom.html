JavaScript and its relationship to HTML and CSS

Alongside HTML and CSS, JavaScript is one of the core building blocks of a website. If we go back and remember the analogy from last week where we had HTML as the bricks (structure) of the house, and CSS as the paint/decoration (the design) of the house, we can also add JavaScript into this analogy. JavaScript adds functionality to the website, so for example, it would be like adding the lights that turn off or on, or adding a microwave that allows you to heat up the food. When you have all three together, you get a structurally solid, well designed, and functional website/house.

Control Flow and Loops

The control flow is the order in which the code is executed. In terms of JavaScript the code is executed from top to bottom. This is the default order of the control flow, however the flow can be interrupted and changed by conditionals and loops. There are many types of loops in JavaScript, but the most common type is a for loop. The for loop works by iterating over the loop a set number of times, and executing the code inside of it. An example of this would be getting ready for a football game. The control flow would dictate the order in which you do things to prepare for this. For example you would make sure you get enough sleep, make a nutritous breakfast, pack your bag, drive to the match and then stretch your muscles. This may be interrupted by loops, for example the pack your bag command could have an if statement that it loops through, checking you have all the required items like shinpads, and if not it tells you to grab them. This proccess repeats until you have everything, and then moves on to the next step

Describe what the DOM is and an example of how you might interact with it

The DOM (document object model) is a tree that is made up of objects. The DOM is created when the website is loaded, and every website has one. An example of this can be found underneath this paragraph. The DOM is a super powerful tool that website developers will be using a lot. Using JavaScript you can access and change any HTML element on the page. This allows you to create some very functional and stylish websites. There are a few methods you can use to tap into the DOM using JavaScript. One of tapping into is by searching for a specific HTML element by ID. You could do this by typing document.getElementByID(‘name of ID’) You could then take this element example change the text of it to green. You can create a CSS class called green-text with the color:green property and apply it by document.getElementByID(‘name of ID’).classList.add(‘green-text’)
